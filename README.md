# How to install ?

<pre>
[ Without Docker]

1.sudo apt-get install nodejs -y
2.npm install && npm start

[ With Docker ]

1.sudo apt-get install docker -y
2.sudo docker build -t sayitmachine ./

</pre>


# How to run ?

<pre>
[ Without Docker ]

Just Type : npm start

[ With Docker ]

Just Type : sudo docker run -it sayitmachine

</pre>
